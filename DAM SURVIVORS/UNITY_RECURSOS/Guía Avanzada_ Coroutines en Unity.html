Guía de Coroutines

Domina el arte de pausar y reanudar el código sin congelar tu juego.

¿Qué es una Coroutine?

Es una función especial que actúa como un "ayudante de cocina". Puedes encargarle una tarea que lleva tiempo (como esperar 5 segundos) y ella la ejecutará en un segundo plano, permitiendo que tu función `Update()` (el "chef principal") siga trabajando sin interrupciones. Esto evita que el juego se congele.

Coroutines vs. Update: ¿Cuándo usar cada una?

  • Usa `Update` para acciones que deben comprobarse en CADA fotograma: leer input, movimiento continuo.
  • Usa una `Coroutine` para secuencias de acciones que ocurren a lo largo del tiempo: generar un enemigo cada 3 segundos, abrir una puerta lentamente, esperar a que una animación termine.

Anatomía y Control de una Coroutine

Anatomía y Lanzamiento

Para crear y usar una coroutine, sigue estos 3 pasos: (importante debes añadir using System.Collections; al script)

// 1. Declara la función con 'IEnumerator'
IEnumerator MiPrimeraCoroutine()
{
    Debug.Log("La coroutine ha empezado.");
    // 2. Usa 'yield return' para pausar
    yield return new WaitForSeconds(2f);
    Debug.Log("La coroutine ha reanudado y terminado.");
}

void Start()
{
    // 3. Lánzala con 'StartCoroutine()'
    StartCoroutine(MiPrimeraCoroutine());
}

Cómo Detener una Coroutine

Es crucial saber cómo detener una coroutine para evitar comportamientos no deseados.

// Hay varias formas de detenerlas:

// Opción A: Guardar la referencia y pararla individualmente
Coroutine miRutinaGuardada = StartCoroutine(MiRutinaInfinita());
StopCoroutine(miRutinaGuardada);

// Opción B: Pararla por su nombre (menos recomendado)
StartCoroutine("MiRutinaInfinita");
StopCoroutine("MiRutinaInfinita");

// Opción C: Parar TODAS las coroutines de este script
StopAllCoroutines();

Tipos de Pausa (`yield return`)

Por Tiempo

Pausa durante un número de segundos. Es la más común.

// Pausa aquí por 5 segundos
yield return new WaitForSeconds(5.0f);

¡Ojo! `WaitForSeconds` se ve afectado por la escala de tiempo (`Time.timeScale`). Si pausas el juego (`Time.timeScale = 0`), esta espera se detendrá. Usa `new WaitForSecondsRealtime(5.0f)` si quieres que la espera ignore la pausa.

Por Fotograma

Pausa la ejecución hasta el siguiente fotograma. La pausa más corta.

// Pausa y espera al siguiente frame
yield return null;

Uso ideal: Para dividir una tarea muy pesada (que podría congelar el juego por un instante) en varios fotogramas, mejorando la fluidez.

Por Condición

Pausa HASTA QUE una condición se cumpla.

// Espera hasta que la vida del jugador sea menor o igual a 0
yield return new WaitUntil(() => vidaJugador <= 0);

Uso ideal: Increíblemente potente para crear secuencias lógicas: esperar a que termine una animación, a que el jugador recoja un objeto, etc.
